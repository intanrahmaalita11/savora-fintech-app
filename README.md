# Savora Fintech App
Savora Fintech App adalah aplikasi berbasis web yang dibuat menggunakan Lovable AI untuk membantu pengguna mengelola tabungan dan pengingat keuangan dengan tampilan yang modern dan responsive.
## Fitur
- Login dan autentikasi pengguna
- Dashboard keuangan
- Manajemen tabungan
- Pengingat keuangan/notifikasi reminder
- Responsive design

## Teknologi yang Digunakan
- JavaScript
- React
- Node.js
- Tailwind CSS
- Supabase

## Cara Menjalankan Project
```bash
npm install
npm run dev
